Elemen Semantik

Elemen semantik adalah elemen-elemen yang menyatakan makna atau tujuan dari elemen itu sendiri. Misalnya tag footer, tag ini digunakan untuk membuat elemen footer atau bagian kaki web. Jangan gunakan tag ini di bagian paling atas, karena maknanya sudah helasa untuk footer.

  • article : membuat elemen artikel.
  • aside : membuat elemen bagian samping.
  • details : membuat elemen detail atau spoiler.
  • figcaption : membuat teks caption pada figure.
  • figure : membuat figur atau gambar pada artikel.
  • footer : membuat elemen bagian kaki dari web.
  • header : membuat kepala kop dari web.
  • main : membuat elemen utama.
  • mark : menandai teks.
  • nav : membuat navigasi.
  • section : membuat bagian artikel.
  • summary : membuat ringkasan artilek / isi spoiler.
  • time : membuat elemen yang menyatakan waktu.

Mengapa harus menggunakan Elemen Semantik?

Salah satu keuntungan menggunakan elemen semantik adalah dokumen HTML kita akan mudah dibaca, baik itu oleh manusia maupun mesin.

Elemen semantik juga bagus untuk SEO. Jadi kalau mau website kita disukai mesin pencari, sebaiknya gunakanlah elemen ini.
