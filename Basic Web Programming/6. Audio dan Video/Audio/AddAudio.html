Menambahkan Audio di HTML

Sebelum adanya HTML 5, audio di HTML ditambahkan dengan program eksternal seperti flash player. Namun, kini HTML sudah punya tag sendiri yakni audio. Tag audio adalah tag untuk membuat audio player. Lalu kita bisa memberikan file audio yang akan diputar dengan tag source.

Susunan codenya seperti dibawah:

        audio controls
            source src="file-audio.mp3" type="audio/mpeg" /
        audio

Format Audio yang didukung

Audio player di HTML tidak mendukung semua jenis format file audio. Berikut ini adalah daftar format file audio yang bisa diputar di HTML.

PCM WAV audio/wav
MP3 MP3 audio/mpeg

Vorbis Ogg audio/ogg

Opus Ogg audio/ogg
Opus WebM audio/webm
FLAC FLAC audio/flac
FLAC Ogg audio/ogg
