Image/gambar akan membuat website kita terlihat menarik. Karena otak kita lebih mudah menyerap informasi dengan visual dibandingkan hanya teks saja. Karena itu, gambar sangatlah penting.


Atribut Image:
1. alt --> Teks alternatif untuk gambar saat gagal ditampilkan;
2. width --> Menentukan lebar gambar;
3. height --> Menentukan tinggi gambar;
4. style --> Menentukan style CSS untuk gambar.

Atribut Style: merupakan atribut untuk menambahkan style CSS pada sebuah elemen. Atribut ini sering digunakan pada gambar untuk memberikan style atau efek tertentu.

Agar gambar bisa ditampilkan sebagai background, maka kita harus menggunakan CSS dengan atribut background-image lalu diisi dengan URL dari gambar.
