Menambahkan Gambar di HTML

Gambar dapat kita tambahkan di HTML dengan menggunakan tag img. Tag ini memiliki artibut wajib, yakni src. Jika kita tidak mengisi atribut src, maka gambar tidak akan ditampilkan.

File gambar yang dituju bisa dalam satu folder dengan file HTML, bisa berbeda folder dengan file HTML, bahkan bisa menggunakan gambar yang ada di internet atau website lain dengan memanggil alamat gambar yang dituju.
