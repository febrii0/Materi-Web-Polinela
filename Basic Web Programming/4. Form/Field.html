Apa itu FIELD?

Field adalah ruas yang dapat diisi dengan data.
Contoh field:
Field memiliki beberapa atribut yang harus diberikan:

  • type : tipe dari field.
  • name : nama dari field yang akan menjadi kunci dan variable di dalam program.
  1. input type='text' : textbox yang bisa menerima inputan text misalkan username atau berupa inputan text yang pendek.
  2. input type='password' : textbox yang menerima inputan text, akan tetapi text yang diinput akan di tampilkan sebagai tanda bintang atau tanda titik, textbox ini biasanya digunakan untuk inputan password.
  3. input type='submit' : inputan ini berupa tombol (button) untuk memproses data inputan dari form.
  4. input type='checkbox' : inputan ini berupa checkbox yang dapat di ceklis oleh user.
  5. input type='radio' : inputan yang berupa radio group, dimana user dapat memilih salah satu dari pilihan yang disediakan, salah satu contoh penggunaan dari radio ini adalah jenis kelamin.

Textarea, Select, dan Option

Dalam HTML 5 terdapat beberapa tag terkait form yaitu:

  1. legend : digunakan untuk menandai kumpulan objek input.
  2. input type='email' : digunakan khusus untuk input data dengan format email.
  3. input type='date' : digunakan khusus untuk input data dengan format tanggal seperti kalender.
