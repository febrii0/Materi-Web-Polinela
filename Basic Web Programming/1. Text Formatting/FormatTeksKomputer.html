Format Teks Untuk Komputer

Teks formatting yang khusus untuk menandai teks yang berasal dari komputer. Berikut ini beberapa tag yang digunakan untuk memformat teks dari komputer:

  1. Tag code : Untuk menandai bagian dari kode program;
  2. Tag samp : Untuk menandai output dari program komputer;
  3. Tag kbd : Untuk menandai tombol keyboard;
  4. Tag var : Untuk menandai sebuah variable;
  5. Tag pre : Untuk preformatting teks.

Contoh:

Perintah Javascript untuk menampilkan teks ke console adalah console.log(). Kita juga bisa menampilkan isi variabel dengan fungsi ini. Misalkan kita punya variabel name, maka kode programnya bisa ditulis seperti ini: 

                var name = "Febriansyah Agung Tirta";
                console.log(name);
Maka hasil outputnya: Febriansyah Agung Tirta

Untuk menjalankan ulang program, lakukan refresh dengan menekan tombol F5 pada keyboard.
