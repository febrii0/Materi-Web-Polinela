Membuat Link / Hyperlink

Link atau Hyperlink adalah elemen HTML yang berfungsi menghubungkan suatu halaman web ke halaman web yang lain. Elemen ini bisa diklik dan nanti akan membuka halaman lain sesuai alamat URL yang diberikan.

Jenis-jenis Link:

Internal Link

Internal Link adalah link yang menuju ke domain atau halaman web itu sendiri.

External Link

External Link adalah link yang menuju domain lain.

Atribut Link:

Atribut Link ini berfungsi untuk menentukan target dari pembukaan link. Diantara lain:

  • _blank : Akan membuka link pada jendela atau tab baru;
  • _self : Akan membuka link pada halaman web itu sendiri (defalit target);
  • _top : Menuju bagian paling atas pada halaman web;
  • _parent : Membuka link pada frame induk halaman web;
  • nama-frame : Akan membuka link pada tag iframe dengan nama tertentu.

Atribut Title

Atribut ini berfungsi untuk membuat tooltips. Tooltips adalah informasi tambahan yang akan tampil saat link disentuh atau saat kita menahan tap di handphone.
