Internal Link adalah link yang menuju ke domain atau halaman web itu sendiri.
Ekternal Link adalah link yang menuju ke domain lain atau halaman web lain.

Atribut Link ini berfungsi untuk menentukan target dari pembukaan link yaitu:
_blank akan membuka link pada jendela atau tab baru;
_self akan membuka link pada halaman itu sendiri (default target);
_top menuju bagian paling atas pada halaman web;
_parent membuka link pada frame induk;
nama-frame akan membuka link pada (iframe) dengan nama tertentu

Atribut Title ini berfungsi untuk membuat tooltips. Tooltips adalah informasi tambahan yang akan tampil saat link disentuh atau saat kita menahan tap di hp.
